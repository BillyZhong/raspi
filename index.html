Raspi
Fast, unopinionated, minimalist Server for the Raspberry Pi

About the Server

Built with the MEAN Stack, the Raspi Server rests on the Raspian Debian/Linux ARMv6 Distribution. Physically the server runs on the Raspberry Pi Microprocessor, which hosts the server. Within the server itself, the Pi runs a MongoDB database for recording users and passwords. In order to process server file systems, Express, a minimalist NodeJS Framework, is used to essentially map the local file system onto the server. To keep the page responsive, AngularJS, a dynamic frontend, renders the pages to allow quick page changes, without sending any actual requests. Angular also manages part of the login verification system. In order to submit commands, a NodeJS library, Socket.io, is used to send asynchronous event-based data, employing a nonblocking structure. In the backend, an asynchronous IO Engine built on Chrome's Javascript Runtime, NodeJS, allows for fast, scalable network applications. In order to manage Linux based commands, the NodeJS child_process class is used to run Linux commands such as Crontab, for time-based commands, Heyu, for controlling appliances, and Pianobar, for controlling music. All of these libraries and frameworks wrap up to create the fast, minimalist server that is Raspi.
